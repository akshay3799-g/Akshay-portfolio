Extracting Text from PowerBI Graphs

By: Akshay 134

This project involved automating the extraction of text from PowerBI graphs and charts to verify the accuracy of displayed values. Since the text within the graphs is not directly accessible, and tooltips dynamically appear with the values upon hovering, traditional methods of extraction were not feasible.

Technologies Used

I utilized PyAutoGUI for cursor movement and screen interaction. The automation process involved systematically moving the cursor across the screen, simulating user interaction. Upon detecting the appearance of the tooltip containing the values, the script captured and collected the text data. This collected data was then compared against expected values to verify the accuracy of the information displayed on the graphs and charts.
